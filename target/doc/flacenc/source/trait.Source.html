fn read_samples<F: Fill>( &mut self, block_size: usize, dest: &mut F, ) -> Result<usize, SourceError>

Reads samples to T: Fill.

Implementation of this function must call either dest.fill_interleaved or dest.fill_le_bytes, and returns the number of per-channel samples read.

§Errors

This function can return SourceError when read is failed.
