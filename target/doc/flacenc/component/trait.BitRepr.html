FLAC components that can be represented in a bit sequence.

fn count_bits(&self) -> usize

fn write<S: BitSink>(&self, dest: &mut S) -> Result<(), OutputError<S>>

Writes the bit sequence to BitSink.

§Errors

This function returns error if self contains an invalid value that does not fit to FLAC’s bitstream format, or if a BitSink method returned an error.
